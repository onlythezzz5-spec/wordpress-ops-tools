# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Contact us directly at: [legal@alicelabs.site](mailto:legal@alicelabs.site)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 72 hours and work with you to resolve the issue.

## Scope

- This project uses the Gemini API — API keys should never be committed to the repository
- Always use `.env.local` for secrets and ensure `.gitignore` covers it
